# About

* Unity package that includes a render feature to add a chromatic aberration effect imitating camera fringing.
* Integrated with URP volume system.
* Unity 6000.3.0f1 or above is required.

# Usage

* Add the chromatic aberration renderer feature to you URP Renderer.
* Enable post-processing both in your camera and URP Renderer.
* Add a volume to your scene and add override: "Custom->Chromatic Aberration".
* Move the intensity parameter.
